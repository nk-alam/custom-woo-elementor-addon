== Installation ==

= Automatic Installation =

1. Go to WordPress Admin ‚Üí Plugins ‚Üí Add New
2. Search for "Custom WooCommerce Product Elementor Addon"
3. Click "Install Now" and then "Activate"

= Manual Installation =

1. Download the plugin zip file
2. Go to WordPress Admin ‚Üí Plugins ‚Üí Add New ‚Üí Upload Plugin
3. Choose the zip file and click "Install Now"
4. Activate the plugin

== Frequently Asked Questions ==

= Does this work with variable products? =

Yes! The plugin is specifically designed to work with WooCommerce variable products and can auto-select the first variation.

= Can I customize the appearance? =

Absolutely! The plugin includes extensive styling options in Elementor including colors, typography, spacing, and layout controls.

= Does it work on mobile devices? =

Yes, the plugin is fully responsive and optimized for mobile devices with touch-friendly interactions.

= Can I add custom product badges? =

Yes, you can add custom badges from the WooCommerce product admin panel. Just enter comma-separated values like "Medium Spicy, Jain Friendly".

= Does the Buy Now button work? =

Yes, the Buy Now button clears the cart, adds the selected product, and redirects to checkout for immediate purchase.

= Is it compatible with caching plugins? =

Yes, the plugin is designed to work with popular caching plugins and uses AJAX for dynamic functionality.

= Can I track analytics? =

Yes, the plugin supports Google Analytics 4, Facebook Pixel, and custom tracking events.
